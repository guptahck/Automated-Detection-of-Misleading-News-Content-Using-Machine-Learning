# 🛡️ Automated Detection of Misleading News Content Using Machine Learning

Welcome to the **Automated Detection of Misleading News Content Using Machine Learning** project! This project is designed to accurately identify and flag misleading news articles using advanced Machine Learning algorithms. 

## ✨ Key Features
- **Accurate Detection:** Utilizes trained Machine Learning models to analyze text patterns.
- **FastAPI Backend:** A robust, high-performance Python backend for rapid AI processing.
- **Next.js Frontend:** A sleek, modern, and professional User Interface.
- **Real-Time Analysis:** Get instant results on news authenticity.

## 🛠️ Technology Stack
- **Backend:** Python, FastAPI, Scikit-Learn, Pandas
- **Frontend:** Next.js, React, TailwindCSS (or Vanilla CSS)
- **Deployment:** Vercel (Frontend), Render/Local (Backend)

## 🚀 How to Run Locally
1. Run `start_project.bat` to launch both the backend and frontend simultaneously.
2. The backend API will be available at `http://localhost:8000`
3. The frontend application will be available at `http://localhost:3000`
